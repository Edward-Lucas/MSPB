MSPB는 오픈소스로 투명하게 공개되어 누구나 코드를 검증할 수 있어 안전합니다.

대시보드 접속 불가

http://127.0.0.1:18080/admin에 접속할 수 없습니다.
대시보드를 사용하려면 아래 프로그램 중 하나를 실행해 주세요.

서버 관리자
mspb-server.exe
클라이언트 사용자
mspb-client.exe

⚠️ IP 밴 기능 경고

IP 밴 기능을 사용하면 모든 사용자의 접속이 차단됩니다.
서버에 연결된 모든 클라이언트가 접속할 수 없게 됩니다.

⚠️ 주의사항
  • • IP 밴은 되돌릴 수 없습니다
  • • 모든 사용자가 서버에 접속할 수 없게 됩니다
  • • 신중하게 사용하세요
